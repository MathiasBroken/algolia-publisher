# Algolia Publisher

### Used Technologies and Frameworks
- Fastify (web framework)
- Typescript
- prettier
- yarn

### How to run
> By default the app uses some values to configure the service. You can change this values in your .env file
- Install dependecies
```
yarn install
```
- Run the app
```
yarn start:dev
```

### Default conf
``` C
APP_KEY=your_app_key
API_KEY=your_api_key
INDEX=your_index
```
